# Make a Picture voting game with nodejs

**How to run:**

    $ git clone https://github.com/marley-nodejs/voting-game
    $ cd voting-game
    $ npm install
    $ npm start

<br/>

**Or you can run app inside docker container**

    $ docker run -it \
    -p 80:8080 \
    --name nodejs-voting-game \
    marley/nodejs-voting-game
